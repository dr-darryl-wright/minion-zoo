Welcome to minion-zoo’s documentation!¶

Example robot classifiers.

This module defines a set of minion objects designed to mimic various classification behaviour. All minions must implement the minion.classify() method to determine how they assign classifications to subjects.

class minion.NoisyMinion(id, name, confusion_matrix)¶

Classifier returns the correct label a specified fraction of the time.

The provided gold standard label is flipped based on the specified noise for this classifier defined in its confusion matrix.

Attributes
idint

unique minion id.

namestr

unique minion name.

confusion_matrixarray-like, shape (2,)

array of confusion matrix elements. First element corresponds to 0 class, second element corresponds to 1 class.
